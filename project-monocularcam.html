Project: Monocular Camera Calibration

  • This assignment aims to familiarize learners with the calibration process for a monocular camera, correcting radial and tangential distortion in captured images. Materials required include: a personal computer, a chessboard calibration board, a monocular camera, and Python 3.10 with libraries like OpenCV, NumPy, ArgParse, glob, and textwrap.
  • The methodology involves working with a dataset of around 30-40 images captured by a GoPro Hero 3+ camera (can be from any other camera). The Python script estimates intrinsic camera parameters and compensates for image distortion; detects corners in calibration images, performs calibration, and outputs camera matrix and lens distortion parameters. After obtaining intrinsic calibration parameters, the script corrects image distortion, saving the corrected images in a folder named undistorted-images. A function is recommended to save camera parameters in a TXT file.
