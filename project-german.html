Project 1: Automated Testing System

This project involved developing an automated testing system using Python and the Kivy framework. The goal was to streamline and automate the manual testing processes for a finance app, ensuring higher efficiency and software reliability.

  • Utilized Python (Kivy) for UI and backend integration.
  • Automated test cases for financial transaction software.
  • Integrated with GitLab for issue tracking and version control.
