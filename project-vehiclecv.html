Project: Vehicle controlled with Computer Vision

  • Objective: Build a small vehicle controlled through a microcontroller that could be programmed with Python to obtain coordinates of movement from a camera on top of the vehicle, controlling the vehicle through computer vision.
  • Materials: Raspberry Pi Zero W, Internet Access, Python 3.11, Python Libraries like OpenCV, Pandas, Matplotlib, Numpy, Web Camera.
  • Operation: With the help of the web camera placed on the ceiling, the algorithm could transform the pixels from the image into coordinates, that could be used to know the position of the vehicle and move it according to the commands set by the user at any given moment to reach the goal.
